Quick Start Setup

Follow this recommended order to set up your business workspace and start using CRM automation.

Checklist
1 Select the active business (e.g. Smart Vision WLL) using the switcher in the header.
2 Link your Google Sheet URL or check the local Excel backup configurations in the Excel Database module.
3 Load or add client profiles under Customers & Directory before running message campaigns.
4 Configure your WhatsApp connection status and verify the indicator says Connected.
5 Go to AI & Workflows to set up chatbot prompts, qualification rules, and automation triggers.
6 Manage customer deals in the Sales Pipeline board, dragging them between stage columns.
7 Issue quotes and invoice records to customers under Sales & Invoices, printing PDF copies.
8 Assign tasks, dispatch resources, and track project completions under Projects & Tasks.

Sidebar Modules Guidance

Every module available in the navigation bar is explained here with step-by-step guidance.

12 Modules

Dashboard

Dashboard Home

Core Modules > Dashboard

Provides business performance charts, real-time KPI card statistics, live operation activity logs, and sub-tabs for Marketing, Finance, and HR summaries.

How to use this module
Steps to analyze performance:
  1. Open the Dashboard view.
  2. Check the KPI cards at the top for Leads, Campaign Delivery, AI Qualification, and Revenue.
  3. Switch between the sub-tabs (Overview, Marketing, Finance, Operations) to access detailed tables.
  4. Review the "Live Event Logs" feed to see recent system events as they happen.

Tip: You can click the "Simulate Inquiry" button to create a mock inbound lead and see how the dashboard charts update.

Sales Pipeline

Kanban Pipeline Board

Core Modules > Sales Pipeline

A visual board where leads are organized into stage columns (New, Contacted, AI Qualified, Proposal Sent, Won/Closed) with color-coded temperatures.

How to use this module
Core operations:
  1. Drag cards between columns to move leads to a new sales stage.
  2. Click any card to open the lead details sidebar, where you can read notes, view chatbot logs, and change assignments.
  3. Click "Add New Lead" to enter a manual lead.

Note: Leads qualified automatically by the WhatsApp Chatbot are pushed directly to the "AI Qualified" column.

Customers & Directory

Customer Profiles Grid

Core Modules > Customers & Directory

The master list of all contacts, displaying names, phones, emails, tags, active pipeline stages, loyalty points, and history notes.

How to use this module
Managing directories:
  1. Use the search bar at the top to filter customers by name or phone.
  2. Click "Add Contact" to insert a customer profile.
  3. Click "Edit" on a customer's row to modify their email, stage, or notes.
  4. Filter contacts based on Labels/Segments (e.g. MEP, paint, HVAC).

Excel Database

Google Sheets & Excel Sync

Core Modules > Excel Database

Connects the local crm.xls/crm.csv files and allows syncing from a remote Google Sheet URL with background auto-sync toggles.

How to use this module
Syncing configurations:
  1. Set your public Google Sheet CSV export link in the URL input field.
  2. Click "Import Google Sheets" to fetch and append new rows.
  3. Toggle "Auto-Sync Google Sheets" to enable automatic background fetching.
  4. If the local file fails to save due to an open Excel session, close Microsoft Excel and click "Retry Saving Database".

WhatsApp Hub

Chat Inbox & Broadcast campaigns

Core Modules > WhatsApp & Chat

A dual-purpose hub containing live simulated messaging conversations, file attachment sharing, templates selection, and bulk campaigns dispatching.

How to use this module
Core features:
  • Chat Inbox: Exchange messages, attach documents/images, and trigger AI drafts in real-time.
  • Broadcasts: Send campaign templates to targeted segments (e.g. Hot Leads, Won Customers).
  • Templates: Write custom text templates with placeholders.

Email Mailbox

Smart Mailbox Suite

Core Modules > Email & Mailbox

Simulates email exchanges with clients, features drag & drop file pickers, template insertions, and custom SMTP server configuration settings.

How to use this module
Sending emails:
  1. Go to Email view and click "Compose Email".
  2. Choose a recipient, write the subject/body, and drag in attachments.
  3. Click "Send Email".
  4. Check the "SMTP Settings" tab to configure your Gmail/Outlook SMTP authentication parameters.

AI Automation

Automation Studio & Chatbot Tuning

Core Modules > AI & Workflows

Tune chatbot behavior (system prompts, temperature, OpenAI models) and create automation rules linked to system triggers.

How to use this module
Creating automation rules:
  1. Select a Trigger (e.g. Lead stage changes, Invoice Paid).
  2. Define conditions (e.g. Lead temperature is Hot).
  3. Configure actions (e.g. Send WhatsApp template, trigger email draft).
  4. Save and test the rules dynamically.

Sales & Invoices

Invoices & Quotations Suite

Core Modules > Sales & Invoices

Generate quotes and invoices, manage product lists, track invoice payment status, and review printable invoice styles.

How to use this module
Invoicing:
  1. Go to Sales view and select the Invoice or Quotation sub-tab.
  2. Click "Create Document" and choose a customer.
  3. Add product items, tax rates, and saving terms.
  4. Click "Save" and click "Print/PDF" to view the document.

Projects & Tasks

Collaboration Task Board

Core Modules > Projects & Tasks

A tasks dashboard divided into To Do, In Progress, Review, and Completed columns, displaying project assignees and deadlines.

How to use this module
Task management:
  1. Add a task, assigning priority (High, Medium, Low) and due dates.
  2. Drag task blocks across progress stages as milestones are reached.
  3. Toggle follow-up reminders to receive email notifications.

Troubleshooting FAQ

Solutions to common errors encountered during local CRM execution.

Issues & Fixes

Database Lock: EBUSY Error

Occurs when trying to import or save customer edits while `crm.xls` is opened inside Microsoft Excel on your computer.

How to fix this
Steps to resolve:
  1. Save and close the `crm.xls` spreadsheet inside Microsoft Excel.
  2. Go back to the CRM browser interface.
  3. Click the "Retry Saving Database" button in the red alert banner or Excel Database tab.
  4. The CRM server will write the changes immediately.

Google Sheet Import Empty Row

Occurs when the synced spreadsheet contains empty lines, missing values, or when Google returns only header names.

How to fix this
Checks to perform:
  • Verify that you have actually typed contact data rows below the header row in your Google Sheet.
  • Make sure your Sheet is set to "Anyone with the link can view".
  • Confirm that the URL is a direct CSV export URL. If needed, re-copy the URL from your browser.

Google Sheet Import Unauthorized (401)

Occurs when the Google Sheet is private/restricted and the CRM cannot access it without authorization.

How to fix this
Steps to resolve:
  1. Open your Google Sheet in a web browser.
  2. Click the blue Share button in the top right corner.
  3. Under "General access", change the status from "Restricted" to Anyone with the link can view.
  4. Make sure the role is set to "Viewer".
  5. Copy the URL from your browser address bar and paste it in the CRM import field again.

WhatsApp Campaign Not Sending

Occurs when campaign messages stay in queue and do not reach the simulated chat list.

How to fix this
Resolution:
  1. Check the header bar and verify the WhatsApp status dot is active green (Connected).
  2. If disconnected, go to Settings > WhatsApp Setup and click "Connect Simulator".
  3. Verify that the target phone numbers are formatted correctly with Qatari country codes (e.g. +974).
